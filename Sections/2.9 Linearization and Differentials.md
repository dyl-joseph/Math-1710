# 2.9 Linearization and Differentials

> [!info] Section 2.9 — Linearization and Differentials
> Covered on Feb 16. Stewart Calculus 9e, Section 2.9.

## Overview

The tangent line at a point gives the best linear approximation to a function near that point. This section formalizes that idea with **linearization** and introduces **differentials** as a tool for estimating changes.

See [[Linear Approximation]] for expanded treatment, standard linearizations, and error propagation.

## Linearization

> [!def] Linearization of $f$ at $a$
> $$L(x) = f(a) + f'(a)(x - a)$$
> The **linear approximation:** $f(x) \approx L(x)$ for $x$ near $a$.

**Procedure to approximate $f(b)$:**
1. Choose $a$ near $b$ where $f(a)$ and $f'(a)$ are easy to compute
2. Write $L(x) = f(a) + f'(a)(x - a)$
3. Evaluate $L(b)$

> [!example] Example 1 — Approximate $\sqrt{9.1}$
> Let $f(x) = \sqrt{x}$, choose $a = 9$ (perfect square nearby).
>
> $f(9) = 3$, $f'(x) = \frac{1}{2\sqrt{x}}$, $f'(9) = \frac{1}{6}$
>
> $L(x) = 3 + \frac{1}{6}(x - 9)$
>
> $\sqrt{9.1} \approx 3 + \frac{1}{6}(0.1) = 3 + 0.01\overline{6} \approx 3.0167$
>
> Actual: $3.01662...\,$ Very accurate.

> [!example] Example 2 — Approximate $\cos(0.05)$
> $f(x) = \cos x$, $a = 0$. $f(0) = 1$, $f'(0) = 0$.
>
> $L(x) = 1 + 0 \cdot x = 1$
>
> $\cos(0.05) \approx 1$. Actual: $0.99875...\,$
>
> The approximation is decent but not great — we could improve by noting $\cos x \approx 1 - \frac{x^2}{2}$ (quadratic approximation, covered later).

> [!example] Example 3 — Linearization of $(1+x)^n$
> $f(x) = (1+x)^n$, $a = 0$.
>
> $f(0) = 1$, $f'(x) = n(1+x)^{n-1}$, $f'(0) = n$
>
> $$\boxed{(1+x)^n \approx 1 + nx \quad \text{for } x \text{ near } 0}$$
>
> This is one of the most useful approximations in all of applied math.

## Differentials

> [!def] Differential
> If $y = f(x)$ is differentiable, define:
> - $dx = \Delta x$ (any real number — an independent variable)
> - $dy = f'(x)\,dx$

The differential $dy$ represents the change in $y$ **along the tangent line**, while $\Delta y = f(x+dx) - f(x)$ is the actual change. When $dx$ is small, $dy \approx \Delta y$.

> [!example] Example 4 — Computing a Differential
> $y = x^2 + 3x$. Find $dy$.
>
> $dy = (2x + 3)\,dx$
>
> At $x = 1$, $dx = 0.05$: $dy = 5(0.05) = 0.25$
>
> Actual: $\Delta y = f(1.05) - f(1) = (1.1025 + 3.15) - (1 + 3) = 0.2525$. Close!

> [!example] Example 5 — Using Differentials to Estimate
> Use differentials to estimate $\ln(1.03)$.
>
> $y = \ln x$, $x = 1$, $dx = 0.03$.
>
> $dy = \frac{1}{x}\,dx = \frac{1}{1}(0.03) = 0.03$
>
> So $\ln(1.03) \approx \ln(1) + 0.03 = 0 + 0.03 = 0.03$.
>
> Actual: $0.02956...\,$ Good estimate.

> [!example] Example 6 — Error Estimation
> The radius of a circle is measured as $r = 15$ cm with possible error $\pm 0.2$ cm. Estimate the maximum error in the computed area.
>
> $A = \pi r^2$, so $dA = 2\pi r\,dr = 2\pi(15)(0.2) = 6\pi \approx 18.85$ cm²
>
> Relative error: $\frac{dA}{A} = \frac{2\pi r\,dr}{\pi r^2} = \frac{2\,dr}{r} = \frac{2(0.2)}{15} \approx 2.67\%$

## Differentials vs $\Delta$

| Symbol | Meaning | Exact? |
|---|---|---|
| $\Delta x = dx$ | Change in $x$ | Exact (we choose it) |
| $\Delta y$ | Actual change in $y$: $f(x+\Delta x) - f(x)$ | Exact |
| $dy$ | Tangent-line change: $f'(x)\,dx$ | Approximation |

As $dx \to 0$, $\frac{\Delta y - dy}{dx} \to 0$, so the approximation gets better.

## Tips

> [!tip] Key Points
> - **Choosing $a$:** Pick a point near $x$ where $f$ and $f'$ are easy to compute (perfect squares, cubes, nice trig values).
> - **Memorize** $(1+x)^n \approx 1+nx$ — it appears constantly in physics and engineering.
> - **Differentials for error:** If $x$ has error $dx$, then $f(x)$ has error $\approx dy = f'(x)\,dx$. Relative error is $\frac{dy}{y}$.
> - **Don't overuse:** Linear approximation only works well near $a$. For $x$ far from $a$, the error can be large.
