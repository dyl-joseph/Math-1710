# 2.6 Implicit Differentiation

> [!info] Section 2.6 — Implicit Differentiation
> Covered on Feb 3. Stewart Calculus 9e, Section 2.6.

## Overview

So far we've differentiated functions written **explicitly** as $y = f(x)$. Many important curves are defined **implicitly** by an equation relating $x$ and $y$ (e.g., $x^2 + y^2 = 25$). Implicit differentiation lets us find $\frac{dy}{dx}$ without solving for $y$.

## Explicit vs Implicit

| Type | Example | Can isolate $y$? |
|---|---|---|
| Explicit | $y = x^3 - 2x$ | Already done |
| Implicit (solvable) | $x^2 + y = 1$ | Yes: $y = 1 - x^2$ |
| Implicit (hard/impossible) | $x^3 + y^3 = 6xy$ | Not easily |

## The Method

> [!def] Implicit Differentiation — Procedure
> 1. **Differentiate** both sides with respect to $x$
> 2. Use the **chain rule** on every $y$-term: $\frac{d}{dx}[(\text{stuff with } y)] = (\text{derivative w.r.t. } y) \cdot \frac{dy}{dx}$
> 3. Use **product rule** wherever $x$ and $y$ are multiplied
> 4. **Collect** all $\frac{dy}{dx}$ terms on one side, everything else on the other
> 5. **Factor out** $\frac{dy}{dx}$ and **solve**

See [[Implicit Differentiation]] for deeper treatment and logarithmic differentiation.

## Worked Examples

> [!example] Example 1 — Circle
> Find $\frac{dy}{dx}$ for $x^2 + y^2 = 25$.
>
> Differentiate: $2x + 2y\frac{dy}{dx} = 0$
>
> Solve: $\frac{dy}{dx} = -\frac{x}{y}$
>
> At $(3, 4)$: slope $= -\frac{3}{4}$. Tangent line: $y - 4 = -\frac{3}{4}(x - 3)$.

> [!example] Example 2 — Folium of Descartes
> Find $\frac{dy}{dx}$ for $x^3 + y^3 = 6xy$.
>
> Differentiate: $3x^2 + 3y^2\frac{dy}{dx} = 6y + 6x\frac{dy}{dx}$
>
> Collect: $3y^2\frac{dy}{dx} - 6x\frac{dy}{dx} = 6y - 3x^2$
>
> Factor: $\frac{dy}{dx}(3y^2 - 6x) = 6y - 3x^2$
>
> $$\frac{dy}{dx} = \frac{6y - 3x^2}{3y^2 - 6x} = \frac{2y - x^2}{y^2 - 2x}$$

> [!example] Example 3 — Involving Trig
> Find $\frac{dy}{dx}$ for $\sin(x + y) = y^2\cos x$.
>
> Differentiate (chain rule on left, product rule on right):
> $$\cos(x+y)\left(1 + \frac{dy}{dx}\right) = 2y\frac{dy}{dx}\cos x - y^2\sin x$$
>
> Expand left side: $\cos(x+y) + \cos(x+y)\frac{dy}{dx} = 2y\cos x \cdot \frac{dy}{dx} - y^2\sin x$
>
> Collect $\frac{dy}{dx}$:
> $$\frac{dy}{dx}[\cos(x+y) - 2y\cos x] = -y^2\sin x - \cos(x+y)$$
>
> $$\frac{dy}{dx} = \frac{-y^2\sin x - \cos(x+y)}{\cos(x+y) - 2y\cos x}$$

> [!example] Example 4 — Tangent Line
> Find the tangent line to $x^2 + xy + y^2 = 7$ at $(1, 2)$.
>
> Differentiate: $2x + y + xy' + 2yy' = 0$
>
> Solve: $y'(x + 2y) = -2x - y$, so $y' = \frac{-2x - y}{x + 2y}$
>
> At $(1,2)$: $y' = \frac{-2-2}{1+4} = -\frac{4}{5}$
>
> Tangent: $y - 2 = -\frac{4}{5}(x - 1)$

> [!example] Example 5 — Second Derivative
> Find $\frac{d^2y}{dx^2}$ for $x^2 + y^2 = 25$.
>
> From Example 1: $\frac{dy}{dx} = -\frac{x}{y}$
>
> Differentiate again (quotient rule):
> $$\frac{d^2y}{dx^2} = -\frac{y - x\frac{dy}{dx}}{y^2} = -\frac{y - x\left(-\frac{x}{y}\right)}{y^2} = -\frac{y^2 + x^2}{y^3} = -\frac{25}{y^3}$$

> [!example] Example 6 — Power Rule for Rational Exponents
> Prove $\frac{d}{dx}(x^{p/q}) = \frac{p}{q}x^{p/q - 1}$ using implicit differentiation.
>
> Let $y = x^{p/q}$, so $y^q = x^p$.
>
> Differentiate: $qy^{q-1}\frac{dy}{dx} = px^{p-1}$
>
> Solve: $\frac{dy}{dx} = \frac{px^{p-1}}{qy^{q-1}} = \frac{px^{p-1}}{q(x^{p/q})^{q-1}} = \frac{px^{p-1}}{qx^{p(q-1)/q}} = \frac{p}{q}x^{p-1 - p + p/q} = \frac{p}{q}x^{p/q - 1}$ $\blacksquare$

## Tips

> [!tip] Common Mistakes to Avoid
> - **Forgetting $\frac{dy}{dx}$ on $y$-terms:** Every time you differentiate something containing $y$, the chain rule gives a factor of $\frac{dy}{dx}$.
> - **Forgetting the product rule:** When $x$ and $y$ are multiplied (like $xy$), you need the product rule: $\frac{d}{dx}(xy) = y + x\frac{dy}{dx}$.
> - **Plugging in values too early:** Find $\frac{dy}{dx}$ as a general expression first, *then* substitute the point.
> - **Not checking the point is on the curve:** Before finding a tangent line at $(a,b)$, verify that $(a,b)$ satisfies the equation.
