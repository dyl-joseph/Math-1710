# 3.9 Antiderivatives

> [!info] Section 3.9 ‚Äî Antiderivatives
> Covered on Mar 23 and Mar 30. Stewart Calculus 9e, Section 3.9.

## Overview

An antiderivative reverses differentiation. If $F'(x) = f(x)$, then $F$ is an antiderivative of $f$. This is the bridge from differential calculus to integral calculus.

## Definition

> [!def] Antiderivative
> A function $F$ is an **antiderivative** of $f$ on an interval $I$ if $F'(x) = f(x)$ for all $x \in I$.

> [!thm] General Antiderivative
> If $F$ is an antiderivative of $f$ on $I$, then the **most general antiderivative** is $F(x) + C$, where $C$ is an arbitrary constant.
>
> (By MVT: if $F' = G'$ on $I$, then $F - G$ is constant.)

The "$+ C$" is essential ‚Äî never forget it.

## Antiderivative Rules

See [[Antiderivative Rules]] for the complete table. Key formulas:

$$\int x^n\,dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)$$

$$\int e^x\,dx = e^x + C$$

$$\int \cos x\,dx = \sin x + C, \qquad \int \sin x\,dx = -\cos x + C$$

$$\int \sec^2 x\,dx = \tan x + C$$

## Worked Examples

> [!example] Example 1 ‚Äî Polynomial
> $$\int (4x^3 - 6x^2 + 2)\,dx = x^4 - 2x^3 + 2x + C$$

> [!example] Example 2 ‚Äî Rewrite Then Integrate
> $$\int \left(\sqrt{x} + \frac{3}{x^2}\right)\,dx = \int \left(x^{1/2} + 3x^{-2}\right)\,dx = \frac{2}{3}x^{3/2} - \frac{3}{x} + C$$

> [!example] Example 3 ‚Äî Trig
> $$\int (2\sin x - 5\cos x)\,dx = -2\cos x - 5\sin x + C$$

> [!example] Example 4 ‚Äî Expand First
> $$\int \frac{(1 + \sqrt{x})^2}{\sqrt{x}}\,dx = \int \frac{1 + 2\sqrt{x} + x}{\sqrt{x}}\,dx = \int (x^{-1/2} + 2 + x^{1/2})\,dx$$
> $$= 2\sqrt{x} + 2x + \frac{2}{3}x^{3/2} + C$$

## Initial Value Problems (IVPs)

Often we're given $f'(x)$ and an initial condition $f(a) = b$ to find the specific antiderivative (determine $C$).

> [!example] Example 5 ‚Äî IVP
> Find $f(x)$ if $f'(x) = 6x^2 - 4x + 3$ and $f(1) = 2$.
>
> $f(x) = 2x^3 - 2x^2 + 3x + C$
>
> $f(1) = 2 - 2 + 3 + C = 3 + C = 2 \implies C = -1$
>
> $f(x) = 2x^3 - 2x^2 + 3x - 1$

> [!example] Example 6 ‚Äî Second-Order IVP
> Find $f(x)$ if $f''(x) = 12x$, $f'(0) = -3$, $f(0) = 5$.
>
> $f'(x) = 6x^2 + C_1$. $f'(0) = C_1 = -3$, so $f'(x) = 6x^2 - 3$.
>
> $f(x) = 2x^3 - 3x + C_2$. $f(0) = C_2 = 5$.
>
> $f(x) = 2x^3 - 3x + 5$

## Physics Application: Motion

Given acceleration, find velocity and position using antiderivatives + initial conditions.

> [!example] Example 7 ‚Äî Free Fall
> A ball is thrown upward from 80 ft with initial velocity 64 ft/s. Find $s(t)$.
>
> $a(t) = -32$ ft/s¬≤ (gravity).
>
> $v(t) = -32t + C_1$. $v(0) = 64 \implies C_1 = 64$. So $v(t) = -32t + 64$.
>
> $s(t) = -16t^2 + 64t + C_2$. $s(0) = 80 \implies C_2 = 80$.
>
> $s(t) = -16t^2 + 64t + 80$
>
> Max height: $v(t) = 0 \implies t = 2$ s, $s(2) = 144$ ft.
>
> Hits ground: $s(t) = 0 \implies -16t^2 + 64t + 80 = 0 \implies t = 5$ s.

## Tips

> [!tip] Key Points
> - **Never forget $+ C$** on indefinite integrals. Points will be deducted.
> - **Rewrite** before integrating: convert roots to fractional powers, divide out fractions, expand products.
> - **Check by differentiating.** If $\frac{d}{dx}[F(x)] = f(x)$, you're right.
> - **No product/chain rule** for antiderivatives (yet ‚Äî substitution in 4.5 handles the chain rule case).
> - **IVPs require two steps:** antidifferentiate, then use the condition to find $C$.
