# 5.5 Average Value of a Function

> [!info] Section 5.5 — Average Value of a Function
> Covered on Apr 27. Stewart Calculus 9e, Section 5.5.

## Overview

Just as we average finitely many numbers by adding and dividing, we can average a continuous function over an interval using integration.

## Definition

> [!def] Average Value
> The **average value** of $f$ on $[a, b]$ is
> $$f_{\text{avg}} = \frac{1}{b - a}\int_a^b f(x)\,dx$$

**Intuition:** The average value is the height of the rectangle with base $[a, b]$ whose area equals the area under $f$.

## The Mean Value Theorem for Integrals

> [!thm] Mean Value Theorem for Integrals
> If $f$ is continuous on $[a, b]$, then there exists $c \in [a, b]$ such that
> $$f(c) = f_{\text{avg}} = \frac{1}{b-a}\int_a^b f(x)\,dx$$

This guarantees $f$ actually **attains** its average value at some point.

## Worked Examples

> [!example] Example 1 — Polynomial
> Find the average value of $f(x) = x^2$ on $[0, 3]$.
>
> $$f_{\text{avg}} = \frac{1}{3-0}\int_0^3 x^2\,dx = \frac{1}{3}\left[\frac{x^3}{3}\right]_0^3 = \frac{1}{3}\cdot 9 = 3$$
>
> Find $c$ where $f(c) = 3$: $c^2 = 3 \implies c = \sqrt{3} \approx 1.73 \in [0, 3]$. ✓

> [!example] Example 2 — Trig
> Average value of $f(x) = \sin x$ on $[0, \pi]$.
>
> $$f_{\text{avg}} = \frac{1}{\pi}\int_0^{\pi}\sin x\,dx = \frac{1}{\pi}[-\cos x]_0^{\pi} = \frac{1}{\pi}(1 + 1) = \frac{2}{\pi} \approx 0.637$$

> [!example] Example 3 — Temperature
> The temperature (°F) $t$ hours after midnight is $T(t) = 50 + 14\sin\left(\frac{\pi t}{12}\right)$.
>
> Average temperature from 9 AM ($t = 9$) to 9 PM ($t = 21$):
>
> $$T_{\text{avg}} = \frac{1}{12}\int_9^{21}\left[50 + 14\sin\left(\frac{\pi t}{12}\right)\right]dt$$
>
> $$= \frac{1}{12}\left[50t - 14\cdot\frac{12}{\pi}\cos\left(\frac{\pi t}{12}\right)\right]_9^{21}$$
>
> $$= \frac{1}{12}\left[(1050 - \frac{168}{\pi}\cos\frac{21\pi}{12}) - (450 - \frac{168}{\pi}\cos\frac{9\pi}{12})\right]$$
>
> $\cos\frac{7\pi}{4} = \frac{\sqrt{2}}{2}$, $\cos\frac{3\pi}{4} = -\frac{\sqrt{2}}{2}$
>
> $$= \frac{1}{12}\left[600 - \frac{168}{\pi}\left(\frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2}\right)\right] = \frac{1}{12}\left[600 - \frac{168\sqrt{2}}{\pi}\right] \approx \frac{600 - 75.6}{12} \approx 43.7°\text{F}$$

> [!example] Example 4 — Linear Function
> Average value of $f(x) = 3x + 1$ on $[1, 5]$.
>
> $$f_{\text{avg}} = \frac{1}{4}\int_1^5 (3x+1)\,dx = \frac{1}{4}\left[\frac{3x^2}{2} + x\right]_1^5 = \frac{1}{4}\left[\frac{75}{2}+5 - \frac{3}{2}-1\right] = \frac{1}{4}(40) = 10$$
>
> Check: for a linear function, the average is the midpoint value: $f(3) = 10$. ✓

## Tips

> [!tip] Key Points
> - **Don't forget the $\frac{1}{b-a}$ factor** — the most common mistake.
> - **For linear functions,** the average value equals $f$ evaluated at the midpoint of the interval.
> - **The MVT for Integrals** guarantees a $c$ exists but doesn't tell you how to find it (solve $f(c) = f_{\text{avg}}$).
> - **Physical interpretation:** average velocity = $\frac{\text{displacement}}{\text{time}} = \frac{1}{b-a}\int_a^b v(t)\,dt$.
> - **Units:** average value has the same units as $f$, not $f \times x$.
