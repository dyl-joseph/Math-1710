# 5.1 Areas Between Curves

> [!info] Section 5.1 — Areas Between Curves
> Covered on Apr 14. Stewart Calculus 9e, Section 5.1.

## Overview

To find the area between two curves, integrate the difference of the "top" and "bottom" functions.

## The Formula

> [!thm] Area Between Curves
> If $f(x) \ge g(x)$ on $[a, b]$, then the area between $y = f(x)$ and $y = g(x)$ is
> $$A = \int_a^b [f(x) - g(x)]\,dx$$
>
> More generally (if curves cross):
> $$A = \int_a^b |f(x) - g(x)|\,dx$$

**Key:** Always subtract **bottom from top** (or split where they cross).

## Procedure

> [!def] Steps
> 1. **Sketch** the curves (or at least find intersection points)
> 2. **Find intersections** by setting $f(x) = g(x)$
> 3. **Determine which is on top** in each interval (plug in a test point)
> 4. **Integrate** $(\text{top}) - (\text{bottom})$ over each interval
> 5. **Add** the areas if the curves cross

## Worked Examples

> [!example] Example 1 — Parabola and Line
> Find the area between $y = x^2$ and $y = x + 2$.
>
> Intersections: $x^2 = x + 2 \implies x^2 - x - 2 = 0 \implies (x-2)(x+1) = 0$, so $x = -1, 2$.
>
> For $x \in [-1, 2]$: line is above parabola (test $x = 0$: $2 > 0$).
>
> $$A = \int_{-1}^{2} [(x+2) - x^2]\,dx = \int_{-1}^{2}(x + 2 - x^2)\,dx$$
> $$= \left[\frac{x^2}{2} + 2x - \frac{x^3}{3}\right]_{-1}^{2} = \left(2 + 4 - \frac{8}{3}\right) - \left(\frac{1}{2} - 2 + \frac{1}{3}\right)$$
> $$= \frac{10}{3} - \left(-\frac{7}{6}\right) = \frac{10}{3} + \frac{7}{6} = \frac{27}{6} = \frac{9}{2}$$

> [!example] Example 2 — Curves That Cross
> Area between $y = \sin x$ and $y = \cos x$ from $0$ to $\pi$.
>
> They cross at $\sin x = \cos x \implies x = \frac{\pi}{4}$.
>
> $[0, \frac{\pi}{4}]$: $\cos x \ge \sin x$. $[\frac{\pi}{4}, \pi]$: $\sin x \ge \cos x$ (check test points).
>
> $$A = \int_0^{\pi/4}(\cos x - \sin x)\,dx + \int_{\pi/4}^{\pi}(\sin x - \cos x)\,dx$$
> $$= [\sin x + \cos x]_0^{\pi/4} + [-\cos x - \sin x]_{\pi/4}^{\pi}$$
> $$= (\sqrt{2} - 1) + [(1 - 0) - (-\sqrt{2})] = (\sqrt{2} - 1) + (1 + \sqrt{2}) = 2\sqrt{2}$$

> [!example] Example 3 — Between Two Parabolas
> Area between $y = x^2$ and $y = 2x - x^2$.
>
> $x^2 = 2x - x^2 \implies 2x^2 - 2x = 0 \implies x = 0, 1$
>
> Top: $2x - x^2$ (test $x = 0.5$: $0.75 > 0.25$).
>
> $$A = \int_0^1 [(2x - x^2) - x^2]\,dx = \int_0^1 (2x - 2x^2)\,dx = \left[x^2 - \frac{2x^3}{3}\right]_0^1 = 1 - \frac{2}{3} = \frac{1}{3}$$

## Integrating with Respect to $y$

Sometimes it's easier to integrate horizontally:

> [!thm] Area Using Horizontal Slices
> If the region is bounded by $x = f(y)$ (right) and $x = g(y)$ (left) for $c \le y \le d$:
> $$A = \int_c^d [f(y) - g(y)]\,dy$$

> [!example] Example 4 — Horizontal Integration
> Area between $x = y^2$ and $x = y + 2$.
>
> Intersections: $y^2 = y + 2 \implies y = -1, 2$.
>
> Right curve: $x = y + 2$. Left curve: $x = y^2$.
>
> $$A = \int_{-1}^{2}[(y+2) - y^2]\,dy = \left[\frac{y^2}{2} + 2y - \frac{y^3}{3}\right]_{-1}^{2} = \frac{9}{2}$$
>
> (Same as Example 1 — same region, but horizontal integration is simpler here since we don't need to split.)

## Tips

> [!tip] Key Points
> - **Always find intersections first** — they determine your limits of integration.
> - **Sketch or test points** to determine which curve is on top.
> - **If curves cross in the interval**, split the integral at each crossing point.
> - **Consider integrating with respect to $y$** if the region is easier to describe horizontally (e.g., bounded by $x = f(y)$ curves).
> - **Common error:** subtracting in the wrong order gives a negative area. Area is always positive.
