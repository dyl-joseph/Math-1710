# 1.6 Calculating Limits Using the Limit Laws

## Overview

In Sections 1.4 and 1.5, we estimated limits numerically (tables) and graphically. Now we compute them **exactly** using algebra and a toolkit of limit laws. The central question: given $\lim_{x\to a}f(x)$, how do we find the answer without a graph or calculator?

The workflow is:
1. **Try direct substitution.** If it gives a finite number, that's the answer.
2. **If you get $\frac{0}{0}$,** simplify algebraically (factor, rationalize, simplify) and try again.
3. **If algebraic methods don't apply** (e.g., oscillating functions), try the [[The Squeeze Theorem|Squeeze Theorem]].

---

## Basic Limits

Every computation starts from two building blocks:

> [!thm] Basic Limits
> $$\lim_{x\to a}c = c \qquad \text{(constant function)}$$
> $$\lim_{x\to a}x = a \qquad \text{(identity function)}$$

These are intuitively obvious and can be proved rigorously with epsilon-delta.

---

## Limit Laws

The [[Limit Laws]] let us combine basic limits to evaluate limits of any expression built from sums, products, quotients, powers, and roots. Assuming $\lim_{x\to a}f(x)=L$ and $\lim_{x\to a}g(x)=M$ exist:

| Law | Statement |
|---|---|
| **Sum** | $\lim_{x\to a}[f(x)+g(x)] = L+M$ |
| **Difference** | $\lim_{x\to a}[f(x)-g(x)] = L-M$ |
| **Constant Multiple** | $\lim_{x\to a}[cf(x)] = cL$ |
| **Product** | $\lim_{x\to a}[f(x)\cdot g(x)] = L\cdot M$ |
| **Quotient** | $\lim_{x\to a}\dfrac{f(x)}{g(x)} = \dfrac{L}{M}$, provided $M\neq 0$ |
| **Power** | $\lim_{x\to a}[f(x)]^n = L^n$ ($n$ a positive integer) |
| **Root** | $\lim_{x\to a}\sqrt[n]{f(x)} = \sqrt[n]{L}$ ($n$ positive integer; if $n$ even, need $L\geq 0$) |

> [!tip] Key point
> The Quotient Law **requires** $\lim_{x\to a}g(x)\neq 0$. When both numerator and denominator approach $0$, the law does not apply — you have the indeterminate form $\frac{0}{0}$ and must do more work.

---

## Direct Substitution Property

> [!thm] Direct Substitution Property
> If $f$ is a polynomial or rational function and $a$ is in the domain of $f$, then
> $$\lim_{x\to a}f(x) = f(a).$$

This extends to all "well-behaved" functions: trig, exponential, logarithmic, and their compositions (see [[Direct Substitution Property]] for details).

> [!example] Example 1 — Direct substitution on a polynomial
> $$\lim_{x\to -1}(x^3 + 2x^2 - x + 4) = (-1)^3 + 2(-1)^2 - (-1) + 4 = -1+2+1+4 = 6.$$

> [!example] Example 2 — Direct substitution on a rational function
> $$\lim_{x\to 2}\frac{x^2+x-6}{x+1} = \frac{4+2-6}{2+1} = \frac{0}{3} = 0.$$
> Here the denominator is $3\neq 0$, so direct substitution is valid. Note that $\frac{0}{3}=0$ is **not** indeterminate — it's just zero.

## Algebraic Techniques for $\frac{0}{0}$

When direct substitution yields $\frac{0}{0}$, the limit may still exist. The idea: algebraically simplify to cancel the common factor causing $\frac{0}{0}$, then substitute again.

### Technique 1: Factoring and Canceling

> [!example] Example 3 — Factoring
> Evaluate $\displaystyle\lim_{x\to 5}\frac{x^2-5x}{x^2-25}$.
>
> Direct substitution: $\frac{25-25}{25-25}=\frac{0}{0}$.
>
> Factor:
> $$\frac{x^2-5x}{x^2-25} = \frac{x(x-5)}{(x-5)(x+5)} = \frac{x}{x+5} \quad (x\neq 5)$$
>
> Now substitute:
> $$\lim_{x\to 5}\frac{x}{x+5} = \frac{5}{10} = \frac{1}{2}.$$

> [!example] Example 4 — Factoring a cubic
> Evaluate $\displaystyle\lim_{x\to -2}\frac{x^3+8}{x+2}$.
>
> Direct substitution: $\frac{-8+8}{0}=\frac{0}{0}$.
>
> Recall the sum-of-cubes formula: $a^3+b^3=(a+b)(a^2-ab+b^2)$.
>
> $$\frac{x^3+8}{x+2} = \frac{(x+2)(x^2-2x+4)}{x+2} = x^2-2x+4 \quad (x\neq -2)$$
>
> $$\lim_{x\to -2}(x^2-2x+4) = 4+4+4 = 12.$$

### Technique 2: Rationalizing (Conjugate Multiplication)

Used when the expression involves square roots.

> [!example] Example 5 — Rationalizing the numerator
> Evaluate $\displaystyle\lim_{t\to 0}\frac{\sqrt{t^2+9}-3}{t^2}$.
>
> Direct substitution: $\frac{\sqrt{9}-3}{0}=\frac{0}{0}$.
>
> Multiply by the conjugate:
> $$\frac{\sqrt{t^2+9}-3}{t^2}\cdot\frac{\sqrt{t^2+9}+3}{\sqrt{t^2+9}+3} = \frac{(t^2+9)-9}{t^2(\sqrt{t^2+9}+3)} = \frac{t^2}{t^2(\sqrt{t^2+9}+3)} = \frac{1}{\sqrt{t^2+9}+3}$$
>
> $$\lim_{t\to 0}\frac{1}{\sqrt{t^2+9}+3} = \frac{1}{\sqrt{9}+3} = \frac{1}{6}.$$

### Technique 3: Simplifying Complex Fractions

Used when the expression contains fractions within fractions.

> [!example] Example 6 — Complex fraction
> Evaluate $\displaystyle\lim_{h\to 0}\frac{\frac{1}{x+h}-\frac{1}{x}}{h}$.
>
> Direct substitution: $\frac{\frac{1}{x}-\frac{1}{x}}{0}=\frac{0}{0}$.
>
> Combine the fractions in the numerator:
> $$\frac{1}{x+h}-\frac{1}{x} = \frac{x-(x+h)}{x(x+h)} = \frac{-h}{x(x+h)}$$
>
> So:
> $$\frac{\frac{-h}{x(x+h)}}{h} = \frac{-h}{h\cdot x(x+h)} = \frac{-1}{x(x+h)}$$
>
> $$\lim_{h\to 0}\frac{-1}{x(x+h)} = \frac{-1}{x\cdot x} = -\frac{1}{x^2}.$$
>
> (This is actually the derivative of $\frac{1}{x}$ — a preview of Chapter 2!)

## The Squeeze Theorem

When algebraic techniques don't help — typically with oscillating functions — we use the [[The Squeeze Theorem|Squeeze Theorem]].

> [!thm] The Squeeze Theorem
> If $f(x)\leq g(x)\leq h(x)$ near $a$ (except possibly at $a$) and
> $$\lim_{x\to a}f(x) = \lim_{x\to a}h(x) = L,$$
> then $\lim_{x\to a}g(x) = L$.

> [!example] Example 7 — Squeeze Theorem application
> Evaluate $\displaystyle\lim_{x\to 0}x^2\sin\!\left(\frac{1}{x}\right)$.
>
> $\sin(1/x)$ oscillates between $-1$ and $1$, so:
> $$-x^2 \leq x^2\sin\!\left(\frac{1}{x}\right) \leq x^2$$
>
> Since $\lim_{x\to 0}(-x^2) = 0 = \lim_{x\to 0}x^2$, the Squeeze Theorem gives:
> $$\lim_{x\to 0}x^2\sin\!\left(\frac{1}{x}\right) = 0.$$

## Important Trigonometric Limit

> [!thm] Sine over $x$ limit
> $$\lim_{x\to 0}\frac{\sin x}{x} = 1$$

This is proved using the Squeeze Theorem and a geometric area argument on the unit circle. The key inequality is:

$$\cos x \leq \frac{\sin x}{x} \leq 1 \qquad \text{for } 0 < |x| < \frac{\pi}{2}$$

Since $\lim_{x\to 0}\cos x = 1$, the Squeeze Theorem gives the result. See [[The Squeeze Theorem]] for the full proof.

> [!tip] Related limit
> $$\lim_{x\to 0}\frac{\cos x - 1}{x} = 0$$
> Often appears alongside $\sin x/x$ in problems and proofs.

## One-Sided Limits of Piecewise and Absolute-Value Functions

For piecewise-defined functions, evaluate the limit from each side separately using the appropriate piece of the definition.

> [!example] Example 8 — Piecewise function
> Let
> $$f(x) = \begin{cases} x^2+1 & \text{if } x < 1 \\ 3x-1 & \text{if } x \geq 1 \end{cases}$$
>
> **Left-hand limit:** For $x<1$, use $f(x)=x^2+1$:
> $$\lim_{x\to 1^-}f(x) = 1^2+1 = 2.$$
>
> **Right-hand limit:** For $x\geq 1$, use $f(x)=3x-1$:
> $$\lim_{x\to 1^+}f(x) = 3(1)-1 = 2.$$
>
> Since both one-sided limits equal $2$, the two-sided limit exists: $\lim_{x\to 1}f(x)=2$.

> [!example] Example 9 — Absolute value
> Evaluate $\displaystyle\lim_{x\to 0}\frac{|x|}{x}$.
>
> Recall $|x| = \begin{cases} x & \text{if } x>0 \\ -x & \text{if } x<0 \end{cases}$.
>
> **Right:** $\displaystyle\lim_{x\to 0^+}\frac{|x|}{x} = \lim_{x\to 0^+}\frac{x}{x} = 1$.
>
> **Left:** $\displaystyle\lim_{x\to 0^-}\frac{|x|}{x} = \lim_{x\to 0^-}\frac{-x}{x} = -1$.
>
> Since $1\neq -1$, the two-sided limit $\lim_{x\to 0}\frac{|x|}{x}$ **does not exist**.

## Strategy Summary

> [!tip] Limit Evaluation Flowchart
> 1. **Try direct substitution.** If $f(a)$ is defined and finite, done.
> 2. **Got $\frac{0}{0}$?** Try:
>    - Factor and cancel
>    - Multiply by conjugate (rationalize)
>    - Simplify compound fractions
>    - Then substitute again.
> 3. **Got $\frac{k}{0}$ with $k\neq 0$?** The limit is $\pm\infty$ or DNE — check signs from left and right.
> 4. **Oscillating/bounded function times something $\to 0$?** Use the [[The Squeeze Theorem|Squeeze Theorem]].
> 5. **Piecewise function?** Evaluate left-hand and right-hand limits separately.

> [!tip] Common mistakes
> - Confusing $\frac{0}{0}$ (indeterminate, needs work) with $\frac{0}{3}$ (just equals $0$) or $\frac{3}{0}$ (limit is $\pm\infty$ or DNE).
> - Forgetting the condition $M\neq 0$ in the Quotient Law.
> - Canceling $(x-a)$ factors without noting "for $x\neq a$" — the cancellation is valid for the limit because limits ignore the value at $a$ itself.
