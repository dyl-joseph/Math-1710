# 4.4 Indefinite Integrals and the Net Change Theorem

> [!info] Section 4.4 — Indefinite Integrals and the Net Change Theorem
> Covered on Apr 9. Stewart Calculus 9e, Section 4.4.

## Overview

This section introduces the indefinite integral notation for antiderivatives and the Net Change Theorem, which interprets definite integrals as accumulated change.

## Indefinite Integrals

> [!def] Indefinite Integral
> $$\int f(x)\,dx = F(x) + C$$
> where $F'(x) = f(x)$. This is just another notation for "the general antiderivative of $f$."

**Key distinction:**
- $\int_a^b f(x)\,dx$ = **number** (definite integral)
- $\int f(x)\,dx$ = **function + C** (indefinite integral)

## Table of Indefinite Integrals

> [!thm] Key Formulas
> $$\int x^n\,dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)$$
> $$\int \frac{1}{x}\,dx = \ln|x| + C$$
> $$\int e^x\,dx = e^x + C$$
> $$\int \sin x\,dx = -\cos x + C \qquad \int \cos x\,dx = \sin x + C$$
> $$\int \sec^2 x\,dx = \tan x + C \qquad \int \csc^2 x\,dx = -\cot x + C$$
> $$\int \sec x\tan x\,dx = \sec x + C \qquad \int \csc x\cot x\,dx = -\csc x + C$$

## Worked Examples

> [!example] Example 1 — Simplify Then Integrate
> $$\int \frac{t^3 + t}{t^2}\,dt = \int \left(t + t^{-1}\right)\,dt = \frac{t^2}{2} + \ln|t| + C$$

> [!example] Example 2 — Expand
> $$\int (\sin\theta + \cos\theta)^2\,d\theta = \int (1 + 2\sin\theta\cos\theta)\,d\theta = \int (1 + \sin 2\theta)\,d\theta$$
> $$= \theta - \frac{1}{2}\cos 2\theta + C$$

> [!example] Example 3 — Definite Integral
> $$\int_1^9 \frac{x - 1}{\sqrt{x}}\,dx = \int_1^9 (x^{1/2} - x^{-1/2})\,dx = \left[\frac{2}{3}x^{3/2} - 2x^{1/2}\right]_1^9$$
> $$= \left(18 - 6\right) - \left(\frac{2}{3} - 2\right) = 12 + \frac{4}{3} = \frac{40}{3}$$

## The Net Change Theorem

> [!thm] Net Change Theorem
> $$\int_a^b F'(x)\,dx = F(b) - F(a)$$
> The integral of a rate of change gives the **net change**.

This is just FTC Part 2 restated, but the interpretation is powerful:

| Rate | Integral gives |
|---|---|
| $v(t)$ = velocity | $\int_a^b v(t)\,dt$ = displacement (net change in position) |
| $|v(t)|$ = speed | $\int_a^b |v(t)|\,dt$ = total distance traveled |
| $R'(t)$ = flow rate | $\int_a^b R'(t)\,dt$ = total volume |
| $P'(t)$ = growth rate | $\int_a^b P'(t)\,dt$ = net population change |
| $C'(x)$ = marginal cost | $\int_a^b C'(x)\,dx$ = total cost from unit $a$ to $b$ |

> [!example] Example 4 — Displacement vs Distance
> A particle has velocity $v(t) = t^2 - 5t + 4 = (t-1)(t-4)$ for $0 \le t \le 6$.
>
> **Displacement:**
> $$\int_0^6 (t^2 - 5t + 4)\,dt = \left[\frac{t^3}{3} - \frac{5t^2}{2} + 4t\right]_0^6 = 72 - 90 + 24 = 6$$
>
> **Total distance:** $v(t) < 0$ on $(1, 4)$, so split:
> $$\int_0^1 (t^2-5t+4)\,dt + \int_1^4 |t^2-5t+4|\,dt + \int_4^6 (t^2-5t+4)\,dt$$
> $$= \int_0^1(\cdots)\,dt - \int_1^4(t^2-5t+4)\,dt + \int_4^6(\cdots)\,dt$$
> $$= \frac{11}{6} + \frac{9}{2} + \frac{14}{3} = \frac{11 + 27 + 28}{6} = \frac{66}{6} = 11$$

## Tips

> [!tip] Key Points
> - **Displacement ≠ total distance.** Displacement allows cancellation (moving forward then backward). Total distance uses $|v(t)|$.
> - **Don't confuse $\int f$ (indefinite, has $+C$) with $\int_a^b f$ (definite, is a number).**
> - **Rewrite before integrating.** Divide, expand, use identities — you can't integrate products directly.
> - **Net Change Theorem** is the go-to for word problems: "integral of rate = total change."
