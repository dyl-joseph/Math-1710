# 4.5 The Substitution Rule

> [!info] Section 4.5 — The Substitution Rule
> Covered on Apr 13. Stewart Calculus 9e, Section 4.5.

## Overview

The Substitution Rule is the integration counterpart of the Chain Rule. It's the most important technique for evaluating integrals that don't match the basic formulas directly.

## The Rule

> [!thm] The Substitution Rule (Indefinite)
> If $u = g(x)$ is differentiable and $f$ is continuous, then
> $$\int f(g(x)) \cdot g'(x)\,dx = \int f(u)\,du$$
> where $u = g(x)$ and $du = g'(x)\,dx$.

> [!thm] The Substitution Rule (Definite)
> $$\int_a^b f(g(x)) \cdot g'(x)\,dx = \int_{g(a)}^{g(b)} f(u)\,du$$
> **Change the limits** when substituting in a definite integral.

## The Method

> [!def] Procedure
> 1. **Choose $u$** — usually the "inner function" in a composition
> 2. **Compute $du = g'(x)\,dx$**
> 3. **Rewrite** the integral entirely in terms of $u$ and $du$
> 4. **Integrate** in terms of $u$
> 5. **Substitute back** $u = g(x)$ (for indefinite integrals) or **change limits** (for definite integrals)

## How to Choose $u$

> [!tip] Guidelines
> - Look for a **function and its derivative** in the integrand
> - $u$ is usually the **inner part** of a composition
> - Common choices: the expression inside a power, inside a trig function, inside a root, in an exponent
> - If you see $g'(x)$ multiplied by something involving $g(x)$, let $u = g(x)$

## Worked Examples

> [!example] Example 1 — Basic Power
> $$\int (2x+1)^5\,dx$$
>
> $u = 2x+1$, $du = 2\,dx$, so $dx = \frac{du}{2}$
>
> $$= \frac{1}{2}\int u^5\,du = \frac{1}{2}\cdot\frac{u^6}{6} + C = \frac{(2x+1)^6}{12} + C$$

> [!example] Example 2 — Trig Composition
> $$\int \cos(3x)\,dx$$
>
> $u = 3x$, $du = 3\,dx$
>
> $$= \frac{1}{3}\int \cos u\,du = \frac{1}{3}\sin u + C = \frac{1}{3}\sin(3x) + C$$

> [!example] Example 3 — Exponential
> $$\int xe^{x^2}\,dx$$
>
> $u = x^2$, $du = 2x\,dx$, so $x\,dx = \frac{du}{2}$
>
> $$= \frac{1}{2}\int e^u\,du = \frac{1}{2}e^{x^2} + C$$

> [!example] Example 4 — Square Root
> $$\int x\sqrt{1 + x^2}\,dx$$
>
> $u = 1 + x^2$, $du = 2x\,dx$
>
> $$= \frac{1}{2}\int u^{1/2}\,du = \frac{1}{2}\cdot\frac{2}{3}u^{3/2} + C = \frac{1}{3}(1+x^2)^{3/2} + C$$

> [!example] Example 5 — Trig with Power
> $$\int \sin^3 x\cos x\,dx$$
>
> $u = \sin x$, $du = \cos x\,dx$
>
> $$= \int u^3\,du = \frac{u^4}{4} + C = \frac{\sin^4 x}{4} + C$$

> [!example] Example 6 — Definite Integral (Change Limits)
> $$\int_0^1 x^2(x^3 + 1)^4\,dx$$
>
> $u = x^3 + 1$, $du = 3x^2\,dx$. When $x=0$: $u=1$. When $x=1$: $u=2$.
>
> $$= \frac{1}{3}\int_1^2 u^4\,du = \frac{1}{3}\left[\frac{u^5}{5}\right]_1^2 = \frac{1}{3}\left(\frac{32}{5} - \frac{1}{5}\right) = \frac{31}{15}$$

> [!example] Example 7 — Tangent
> $$\int \tan x\,dx = \int \frac{\sin x}{\cos x}\,dx$$
>
> $u = \cos x$, $du = -\sin x\,dx$
>
> $$= -\int \frac{du}{u} = -\ln|u| + C = -\ln|\cos x| + C = \ln|\sec x| + C$$

## Symmetry Shortcuts

> [!thm] Even and Odd Functions
> If $f$ is **even** ($f(-x) = f(x)$): $\int_{-a}^{a} f(x)\,dx = 2\int_0^a f(x)\,dx$
>
> If $f$ is **odd** ($f(-x) = -f(x)$): $\int_{-a}^{a} f(x)\,dx = 0$

> [!example] Example 8 — Symmetry
> $$\int_{-1}^{1} (x^5 + x^3 + x)\,dx = 0$$
> since $x^5 + x^3 + x$ is odd.

## Tips

> [!tip] Key Points
> - **For definite integrals:** either change the limits (recommended) or substitute back and use the original limits. Never mix the two.
> - **If you can't find $du$ exactly:** check if you can account for a constant factor (e.g., $x\,dx = \frac{1}{2}du$ when $u = x^2$).
> - **Not every integral yields to substitution.** If $u$-sub doesn't work, the integral may require a different technique or may not have a closed form.
> - **Check by differentiating** your answer.
