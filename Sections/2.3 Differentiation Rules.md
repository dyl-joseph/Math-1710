# 2.3 Differentiation Rules

## Overview

In Section 2.2, we computed derivatives from the limit definition — a process that works but is tedious. This section develops **shortcut rules** that let us differentiate polynomials, rational functions, root functions, and exponentials quickly and reliably. These rules are the workhorses of calculus.

**Covered over two days (1/27 and 1/29)** because of the number of rules and the practice required to use them fluently.

---

## The Constant Rule

> [!thm] Constant Rule
> If $c$ is a constant, then
> $$\frac{d}{dx}(c) = 0$$

### Proof

From the limit definition:

$$\frac{d}{dx}(c) = \lim_{h\to 0}\frac{c - c}{h} = \lim_{h\to 0}\frac{0}{h} = \lim_{h\to 0}0 = 0 \quad\blacksquare$$

Geometrically: the graph of $y = c$ is a horizontal line, and horizontal lines have slope $0$.

## The Power Rule

> [!thm] Power Rule
> If $n$ is any real number, then
> $$\frac{d}{dx}(x^n) = nx^{n-1}$$

See [[Power Rule]] for proofs at each level of generality (positive integers via the binomial theorem, negative integers via the quotient rule, rationals via implicit differentiation, and all reals via logarithmic differentiation).

> [!example] Example 1: Derivative of a polynomial
> Find $f'(x)$ where $f(x) = 3x^4 - 5x^3 + 2x - 7$.
>
> Apply the power rule term by term (using the constant multiple and sum rules below):
>
> $$f'(x) = 3(4x^3) - 5(3x^2) + 2(1) - 0 = 12x^3 - 15x^2 + 2$$

> [!example] Example 2: Rewrite then differentiate (negative/fractional exponents)
> Find $g'(x)$ where $g(x) = \dfrac{4}{x^2} - 3\sqrt[3]{x} + \sqrt{x^5}$.
>
> **Step 1 — Rewrite** as power functions:
> $$g(x) = 4x^{-2} - 3x^{1/3} + x^{5/2}$$
>
> **Step 2 — Differentiate** term by term:
> $$g'(x) = 4(-2)x^{-3} - 3\cdot\tfrac{1}{3}x^{-2/3} + \tfrac{5}{2}x^{3/2}$$
>
> $$= -8x^{-3} - x^{-2/3} + \tfrac{5}{2}x^{3/2} = \frac{-8}{x^3} - \frac{1}{x^{2/3}} + \frac{5\sqrt{x^3}}{2}$$

> [!tip] Rewrite first!
> Always convert radicals to fractional powers and move terms from the denominator using negative exponents **before** differentiating. This avoids unnecessary quotient rule applications.

## The Constant Multiple Rule

> [!thm] Constant Multiple Rule
> If $c$ is a constant and $f$ is differentiable, then
> $$\frac{d}{dx}[c\,f(x)] = c\,f'(x)$$

### Proof

$$\frac{d}{dx}[cf(x)] = \lim_{h\to 0}\frac{cf(x+h) - cf(x)}{h} = c\,\lim_{h\to 0}\frac{f(x+h)-f(x)}{h} = c\,f'(x) \quad\blacksquare$$

The constant factors out of the limit because it does not depend on $h$.

---

## The Sum and Difference Rules

> [!thm] Sum/Difference Rule
> If $f$ and $g$ are both differentiable, then
> $$\frac{d}{dx}[f(x) + g(x)] = f'(x) + g'(x)$$
> $$\frac{d}{dx}[f(x) - g(x)] = f'(x) - g'(x)$$

### Proof (Sum Rule)

$$\frac{d}{dx}[f(x)+g(x)] = \lim_{h\to 0}\frac{[f(x+h)+g(x+h)] - [f(x)+g(x)]}{h}$$

$$= \lim_{h\to 0}\left[\frac{f(x+h)-f(x)}{h} + \frac{g(x+h)-g(x)}{h}\right] = f'(x) + g'(x) \quad\blacksquare$$

The difference rule follows by combining the sum rule with the constant multiple rule (using $c = -1$).

> [!tip] These rules extend to any finite number of terms
> $$\frac{d}{dx}[f_1(x) + f_2(x) + \cdots + f_n(x)] = f_1'(x) + f_2'(x) + \cdots + f_n'(x)$$
> This is why we can differentiate polynomials **term by term**.

## Differentiating General Polynomials

Combining the power rule, constant multiple rule, and sum rule, we can differentiate any polynomial at sight. If

$$p(x) = a_nx^n + a_{n-1}x^{n-1} + \cdots + a_1x + a_0$$

then

$$p'(x) = na_nx^{n-1} + (n-1)a_{n-1}x^{n-2} + \cdots + a_1$$

## The Product Rule

> [!thm] Product Rule
> If $f$ and $g$ are both differentiable, then
> $$\frac{d}{dx}[f(x)\,g(x)] = f'(x)\,g(x) + f(x)\,g'(x)$$

See [[Product and Quotient Rules]] for the proof from the limit definition, the extended three-factor version, and common pitfalls.

> [!example] Example 3: Product Rule
> Find $\frac{d}{dx}\left[(2x+1)(x^3-x)\right]$.
>
> Let $f = 2x+1$, $g = x^3 - x$.
>
> $$f' = 2, \qquad g' = 3x^2 - 1$$
>
> $$(fg)' = 2(x^3-x) + (2x+1)(3x^2-1)$$
>
> $$= 2x^3-2x + 6x^3-2x+3x^2-1 = 8x^3+3x^2-4x-1$$

## The Quotient Rule

> [!thm] Quotient Rule
> If $f$ and $g$ are differentiable and $g(x)\neq 0$, then
> $$\frac{d}{dx}\left[\frac{f(x)}{g(x)}\right] = \frac{f'(x)\,g(x) - f(x)\,g'(x)}{[g(x)]^2}$$

See [[Product and Quotient Rules]] for the proof and mnemonic.

> [!tip] "Lo d-Hi minus Hi d-Lo over Lo-Lo"
> Denominator = **Lo**, Numerator = **Hi**.
> $$\frac{\text{Lo}\cdot d(\text{Hi}) - \text{Hi}\cdot d(\text{Lo})}{\text{Lo}\cdot\text{Lo}}$$
> The **minus** sign means order matters — always start with the denominator times the derivative of the numerator.

> [!example] Example 4: Quotient Rule
> Differentiate $y = \dfrac{x^2 - 1}{x^2 + 1}$.
>
> $$y' = \frac{(2x)(x^2+1) - (x^2-1)(2x)}{(x^2+1)^2}$$
>
> $$= \frac{2x^3+2x-2x^3+2x}{(x^2+1)^2} = \frac{4x}{(x^2+1)^2}$$

> [!tip] Don't use the quotient rule when you can simplify first
> If the denominator is a single term (a monomial), divide each term of the numerator by it and use the power rule instead. This is faster and less error-prone.

## The Natural Exponential Function

> [!thm] Derivative of $e^x$
> $$\frac{d}{dx}(e^x) = e^x$$

This is a remarkable property: $e^x$ is its own derivative. The number $e\approx 2.71828$ is precisely the base for which this is true.

> [!def] The number $e$
> $e$ is defined as the number such that
> $$\lim_{h\to 0}\frac{e^h - 1}{h} = 1$$
> This limit being exactly $1$ is what makes $\frac{d}{dx}(e^x) = e^x$.

Combined with the constant multiple rule:

$$\frac{d}{dx}(Ce^x) = Ce^x$$

## More Worked Examples

> [!example] Example 5: Combination of rules
> Differentiate $f(x) = x^2e^x + 3e^x$.
>
> For the first term, use the **product rule** with $u = x^2$ and $v = e^x$:
> $$\frac{d}{dx}(x^2e^x) = 2xe^x + x^2e^x$$
>
> For the second term:
> $$\frac{d}{dx}(3e^x) = 3e^x$$
>
> $$f'(x) = 2xe^x + x^2e^x + 3e^x = e^x(x^2+2x+3)$$

> [!example] Example 6: Equation of a tangent line
> Find the equation of the tangent line to $y = \dfrac{2x}{x-1}$ at the point $(2, 4)$.
>
> **Step 1 — Find the derivative** (quotient rule):
> $$y' = \frac{2(x-1) - 2x(1)}{(x-1)^2} = \frac{2x - 2 - 2x}{(x-1)^2} = \frac{-2}{(x-1)^2}$$
>
> **Step 2 — Evaluate at $x = 2$**:
> $$y'(2) = \frac{-2}{(2-1)^2} = \frac{-2}{1} = -2$$
>
> **Step 3 — Point-slope form**:
> $$y - 4 = -2(x - 2) \implies y = -2x + 8$$

> [!example] Example 7: Horizontal tangent lines
> Find all points where $f(x) = x^3 - 6x^2 + 9x + 1$ has a horizontal tangent.
>
> A horizontal tangent means $f'(x) = 0$.
>
> $$f'(x) = 3x^2 - 12x + 9 = 3(x^2 - 4x + 3) = 3(x-1)(x-3)$$
>
> Setting $f'(x) = 0$: $x = 1$ or $x = 3$.
>
> At $x = 1$: $f(1) = 1 - 6 + 9 + 1 = 5$, so the point is $(1, 5)$.
>
> At $x = 3$: $f(3) = 27 - 54 + 27 + 1 = 1$, so the point is $(3, 1)$.
>
> The tangent line is horizontal at $(1, 5)$ and $(3, 1)$.

> [!example] Example 8: Derivative of $\dfrac{e^x}{x^2}$
> Using the quotient rule with $f = e^x$ and $g = x^2$:
>
> $$\frac{d}{dx}\left(\frac{e^x}{x^2}\right) = \frac{e^x\cdot x^2 - e^x\cdot 2x}{x^4} = \frac{e^x(x^2-2x)}{x^4} = \frac{e^x(x-2)}{x^3}$$

## Summary of Differentiation Rules

| Rule | Formula |
|---|---|
| Constant | $\frac{d}{dx}(c) = 0$ |
| Power | $\frac{d}{dx}(x^n) = nx^{n-1}$ |
| Constant Multiple | $\frac{d}{dx}[cf] = cf'$ |
| Sum/Difference | $\frac{d}{dx}[f\pm g] = f'\pm g'$ |
| Product | $\frac{d}{dx}[fg] = f'g + fg'$ |
| Quotient | $\frac{d}{dx}\!\left[\frac{f}{g}\right] = \frac{f'g-fg'}{g^2}$ |
| Natural Exponential | $\frac{d}{dx}(e^x) = e^x$ |
